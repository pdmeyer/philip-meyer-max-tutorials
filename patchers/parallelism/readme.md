Note: This is an UNFROZEN AMXD. To work, it needs access to remote.maxpat, which is located in the Track Mixer Project folder. 

To use this device in Live, download this entire folder, then drag and drop the AMXD file onto a track in Live. You need to make sure that the Track Mixer Project folder is in the same folder as the AMXD file. 

If you want, you can freeze the Max for Live device. Freezing bundles the dependencies with the AMXD file and simplifie sthings a bit. You can do so by editing the device, clicking the snowflake icon at the bottom of the patcher window, and then saving the device.
